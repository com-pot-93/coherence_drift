The process starts when a production order is received. Following this, materials are prepared, and the door is assembled. An exclusive gateway then determines the next step. If the condition is true, an inquiry for a quality check is sent and subsequently received. The quality is then checked, and a quality report is sent and received. Another exclusive gateway then determines whether to fix the door or release it. If the decision is to fix the door, the door is fixed, and the process returns to the first exclusive gateway. If the decision is to release the door, the door is released, and the process returns to the first exclusive gateway. If the condition at the first exclusive gateway is false, the process ends.
