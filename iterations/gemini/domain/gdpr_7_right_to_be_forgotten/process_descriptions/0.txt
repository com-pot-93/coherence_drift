The process starts when a data subject request is received. Then, the data is retrieved and a check is performed to determine if the received data is still relevant. Depending on the outcome of this check, the process follows one of two exclusive paths. If the data is still relevant, an explanation is provided as to why. Alternatively, if the data is not relevant, it is eliminated, and a notification about the elimination is sent. Finally, the process ends after either explaining why the data is still relevant or sending the elimination notification.
