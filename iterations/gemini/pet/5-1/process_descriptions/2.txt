The process starts when a loan request is received. The risk associated with the request is then evaluated. Based on the evaluation, the request is either approved, denied, or sent to an approver web service. If the request is approved, the loan is approved. If the request is denied, the loan is denied. If the request is sent to an approver web service, it is processed by that service. After any of these three paths, feedback is provided to the customer, and the process ends.
