The process starts when a loan request is received. The risk associated with the loan is then evaluated. Based on the evaluation, the process follows one of three paths: the loan is either approved, denied, or sent to an approver web service. If the loan is approved, the loan is approved. If the loan is denied, the loan is denied. If the loan is sent to an approver web service, the loan is processed via the web service. After the loan is either approved, denied, or processed via the web service, feedback is provided to the customer. The process then ends.
