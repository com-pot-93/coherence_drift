The process starts when a loan request is received. The risk associated with the request is then evaluated. Depending on the evaluation, the process follows one of three paths: the loan is either approved, denied, or sent to an approver web service for processing. If the loan is approved, a task to approve the loan is performed. If the loan is denied, a task to deny the loan is performed. If the loan is sent to the approver web service, a task to process the loan via the web service is performed. After the loan is either approved, denied, or processed via the web service, feedback is provided to the customer. The process ends after the feedback is provided.
