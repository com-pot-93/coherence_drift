The process starts when a loan request is received. The risk associated with the request is then evaluated. Based on the evaluation, a decision is made: the loan can be approved, denied, or sent to an approver web service for further processing. If the loan is approved, the loan is approved. If the loan is denied, the loan is denied. If the loan is sent to an approver web service, the loan is processed via the approver web service. After the loan is either approved, denied, or processed via the approver web service, feedback is provided to the customer. The process ends after the feedback is provided.
