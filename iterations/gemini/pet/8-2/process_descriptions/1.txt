The process starts with reporting a vacancy to HR, followed by creating a job description. If clarification is necessary, the department is asked for it. Otherwise, the process proceeds directly. Next, if a condition is met, the job description is submitted. This step is followed by a complex decision-making process involving multiple exclusive gateways. If correction is needed at any of these gateways, the description is corrected, and the process loops back. Otherwise, the process continues forward. Alternatively, if a condition is met, the job description is submitted again, and an escalation event occurs. Finally, the job is posted, and the process ends.
