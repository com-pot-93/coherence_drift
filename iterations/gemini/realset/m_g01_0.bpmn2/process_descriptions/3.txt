The process starts and immediately encounters a decision point. If the condition is true, it proceeds to another decision point. If this second condition is true, a status is sent, and the process moves to another decision point. If this third condition is true, an object is printed before the process merges back. If the third condition is false, the process merges directly. After the merge, the process merges again before returning to the first decision point. If the second condition is false, the process merges directly. After the merge, the process returns to the first decision point. If the first condition is false, the process ends.
