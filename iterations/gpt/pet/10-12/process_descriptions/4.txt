The process begins when EC informs INQ about a change. After receiving this information, INQ notifies IP about the change. Next, IP reviews the proposed change.

At this point, IP decides whether the change can be made. If IP determines that the change can be made, IP confirms the changes to INQ. If IP decides that the change cannot be made, IP rejects the changes to INQ.

After either confirming or rejecting the changes, the process comes to an end.
