The process begins when EC informs INQ about a change in data. After receiving this information, INQ notifies IP about the change. Next, IP reviews the situation to determine whether the data can be changed.

At this point, there are two possible outcomes:

- If IP decides that the data can be changed, IP confirms the changes to INQ.
- If IP decides that the data cannot be changed, IP rejects the changes to INQ.

Regardless of whether the changes are confirmed or rejected, the process then comes to an end.
