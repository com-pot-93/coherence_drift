The process begins when a request for a warrant of possession is received. After receiving this request, the next step is to retrieve the relevant file associated with the Small Claims Tribunal (SCT). Once the file has been retrieved, the SCT warrant of possession is forwarded to the Queensland Police.

Following this, the physical SCT file is stored. The process then involves waiting for a report from the police. When the police report is received, the SCT file is retrieved again. A new SCT document is then attached to this file. After attaching the new document, the updated physical SCT file is stored. This marks the end of the process.
