The process begins when a request for a warrant of possession is received. After receiving this request, the next step is to retrieve the relevant SCT file. Once the SCT file has been retrieved, the SCT warrant is forwarded to the Queensland Police.

Following this, the physical SCT file is stored. The process then involves waiting for a report from the police. When the police report is received, the SCT file is retrieved again.

After retrieving the file, a new SCT document is attached to it. The updated SCT file is then stored. This marks the end of the process.
