The process begins when a request for a warrant of possession is received. After receiving this request, the next step is to retrieve the relevant SCT file. Once the SCT file has been retrieved, the SCT warrant is forwarded to the Queensland Police.

Following this, the physical SCT file is stored. The process then waits until a report is received from the police. When the police report arrives, the SCT file is retrieved again. A new SCT document is then attached to the file. After attaching the new document, the updated SCT file is stored. This marks the end of the process.
