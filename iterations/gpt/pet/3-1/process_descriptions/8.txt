The process begins when a request for a Warrant of Possession is received. After receiving this request, the next step is to retrieve the relevant file associated with the Small Claims Tribunal (SCT). Once the appropriate SCT file has been found, the warrant is then forwarded to the Queensland Police.

Following this, the physical SCT file is stored securely. The process then involves waiting for a report from the police. When the police report is received, the SCT file is retrieved again. At this point, a new document related to the SCT is attached to the file. Finally, the updated SCT file is stored, marking the end of the process.
