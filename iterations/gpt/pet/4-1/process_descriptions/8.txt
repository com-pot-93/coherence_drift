The process begins when a notice is received at the secretarial office. After receiving the notice, the secretarial office requests patient data. The necessary patient data is then obtained. Once the data is available, the doctor is connected to the nursing officer. The nursing officer inquires about the patient, and the information gathered is recorded on a registration form. This completed registration form is then returned to the secretarial office.

Next, the information from the registration form is entered into the information system. Afterward, the information from the form is printed. At this point, a decision is made about whether a file is required for the patient. If a file is required, a patient file is created, and both the form and the printout are stored in the file. If no file is required, this step is skipped.

Regardless of whether a file was created, the process continues with the production of two registration cards. The patient is then added to the list of new notices. A staff meeting is held to assign patients, and the assignments are recorded on an assignment list.

Another decision is made to determine if a medical file is needed. If a medical file is needed, a decision about the medical file is made. If not, this step is skipped. The process then continues with adding the assignment information to the information system. The registration cards are distributed.

Next, it is determined whether a letter needs to be sent to the family doctor. If a letter is needed, it is prepared and sent. If not, this step is skipped. After this, a copy of the patient’s medical file is received. The responsible intake person is informed, and the copy of the medical file is added to the patient file.

A meeting with the patient is planned, and the patient is examined. Any additional observations are recorded in notes, and a copy of these notes is added to the patient file. A standard checklist is also added to the patient file.

At this stage, it is decided whether a second intake is needed. If a second intake is needed, the second intake worker plans the first meeting. If not, this step is skipped. The process continues with typing out recorded tapes, and the typed notes are added to the patient file.

The patient is then put on the list of patients, and this list is provided to the team leader. Finally, a treatment plan is created, and the process ends.
